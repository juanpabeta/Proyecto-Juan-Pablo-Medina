# Convertidor Universal de Unidades - Proyecto Final

Este repositorio contiene el código de mi proyecto final para el curso de python cohorte 5 de DevSeniorCode .


## Descripción del Proyecto

El programa permite al usuario realizar conversiones en tres categorías principales:

1.  **Temperatura:** Permite hacer conversiones de Celsius, Fahrenheit y Kelvin.
2.  **Longitud:** Permite convertir entre múltiples unidades como (mm, cm, m, km, etc.).
3.  **Divisas:** Permite hace conversiones entre distintas divisa del mundo como (USD,EUR,COP;JPY) 

##  Funcionalidades Técnicas

Este código fue construido para que se puede enterder claramente todo el codigo. Las características principales de este codigo son las siguiente:

* **Modularidad:** Uso de funciones(def) para separar el codigo en secciones para que se más facil de entender.
* **Lógica de Decisión:** Implementación de estructuras `if-elif-else` para gestionar el flujo del programa.
* **Limpieza de Datos:** Implementación de métodos como `.strip()` y `.lower()` para evitar errores por entradas del usuario mal formateadas.
* **Validaciones:** Manejo básico de errores cuando se ingresan unidades no reconocidas.
